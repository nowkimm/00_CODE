## 14. 크리에이티브 워크플로우 팁

### 14.1 효율적인 제작 프로세스

```
1. 컨셉 단계
   - 아이디어 스케치 생성
   - 여러 변형 탐색
   - 방향 선택

2. 개발 단계
   - 선택된 컨셉 정교화
   - 디테일 추가
   - 색상/스타일 확정

3. 완성 단계
   - 최종 버전 생성
   - 필요한 포맷으로 조정
   - 시리즈 일관성 확인
```

### 14.2 반복 개선 전략

```
효과적인 수정 요청:
- "캐릭터의 표정을 더 밝게"
- "배경을 좀 더 단순하게"
- "색상을 브랜드 팔레트에 맞게 조정"
- "구도를 왼쪽으로 이동"
- "더 다이나믹한 포즈로"
```

### 14.3 시리즈 일관성 유지

```
시리즈 작업 시:
1. 첫 번째 이미지를 기준으로 설정
2. 스타일 가이드 프롬프트 작성
3. 매번 동일한 기본 프롬프트 사용
4. 참조 이미지로 이전 결과물 활용
5. 변형만 별도로 지시
```
