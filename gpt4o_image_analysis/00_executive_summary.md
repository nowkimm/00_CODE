## ⚠️ 한계점 및 주의사항

### 기술적 한계

1. **텍스트:** 비라틴 문자 불안정, 긴 텍스트 오류
2. **해부학:** 손/손가락 렌더링 문제 지속
3. **정확도:** 인포그래픽/다이어그램 신뢰성 부족
4. **해상도:** 최대 1536x1024, 업스케일 미지원
5. **일관성:** 완벽한 픽셀 단위 복제 불가

### 윤리적 고려

- 저작권: 생존 작가 스타일 복제 제한
- 개인정보: 업로드 이미지 데이터 정책 확인 필요
- 딥페이크: 실제 인물 재현 신중하게 접근
- 편향: 특정 인종/성별 표현 편중 가능성

### 서비스 제한

- 무료: 하루 3회 생성, 일부 스타일 차단
- 유료: API Rate limits, 토큰 한도
